# FitTimer

Timer de intervalos para rutinas con ejercicios, descansos cortos, descansos largos y rondas.

## Uso

Abre `index.html` en el navegador.

Configura:

- `Ejercicio`: segundos activos por repeticion.
- `Descanso corto`: pausa entre repeticiones del mismo ejercicio.
- `Repeticiones`: cantidad de series por ejercicio.
- `Descanso largo`: pausa para cambiar de ejercicio o ronda.
- `Ejercicios`: cantidad de ejercicios por ronda.
- `Rondas`: veces que se repite el bloque completo.

Durante el entrenamiento puedes pausar, reanudar, reiniciar, volver a configurar o poner la pantalla en modo completo.
